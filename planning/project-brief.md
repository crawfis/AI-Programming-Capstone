# Project Brief

*This file is populated by the `capstone-kickoff` skill. Say "Help me brainstorm my capstone project" to get started.*

Once populated, this file will contain:

- **Vision** — what you're building and why
- **Platform & Stack** — language, framework, hosting, visual style
- **Target Audience** — who this is for
- **Constraints** — budget, time, API requirements, accessibility
- **MVP Features** — numbered list with acceptance criteria
- **Stretch Goals** — ordered Should Have / Could Have features
- **Explicit Exclusions** — Won't Have list with reasons
- **Analysis Summary** — which decision frameworks were applied and key findings
- **Toolchain** — ecosystems to install, existing skills to use, custom skills to create
